# 🖥️ Y2K 앨범 소개 페이지

> Windows 98 감성의 Y2K 스타일 앨범 소개 웹페이지

## 📁 파일 구조
📁 프로젝트
├── y2k_site.html # 메인 페이지 (배경 + 그리드)
├── album.html # 앨범 소개 콘텐츠
└── your-y2k-image.png # 배경 이미지
## 🚀 실행 방법 1. 세 파일을 같은 폴더에 넣기 2. `your-y2k-image.png` 를 실제 이미지 파일명으로 교체 3. `y2k_site.html` 브라우저로 열기 ## ✨ 주요 기능 - Y2K 감성 배경 이미지 위에 iframe으로 앨범 페이지 삽입 - 화면 크기에 따라 비율 유지하며 자동 스케일 - CSS 원근감 그리드 배경 - DungGeunMo 폰트 적용 - Windows 98 스타일 UI 디자인 ## 🛠️ 사용 기술 - HTML / CSS - CSS Transform (perspective, rotateX) - iframe srcdoc
